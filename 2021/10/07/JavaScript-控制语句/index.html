JS 中关键字 break 以及 continue 的作用

break：在循环体内遇到 break 直接跳出当前循环

continue：在循环体内遇到 continue 关键字跳出本次循环，继续执行下次循环

for in遍历循环

for in 循环遍历的两种应用场景

第一种应用场景：循环遍历 JS 中的数组【列表】，获取数组内的数据

第二种应用场景：遍历 JS 中的实例化对象，从而获取对象内的所有属性名【属性名全部都是字符串】

for in 循环遍历元组的语法

语法：for（var i in 元组名）{ }

注意：

1：JS 内元组的数据类型和数据的数量不限

2：for in 循环遍历元组，变量 i 获得的只是元组内元素的下标索引，要想获取元组内数据，需要 元组名[获取到的数据下标索引]

for in 循环遍历 Object 对象的语法

语法：for（var i in Object实例化的对象）{ …… }

功能：获取实例化对象内所有的属性名称，从而根据属性名称获取实例化对象属性值

注意：for in 循环遍历 Object 对象获取到对象的属性名是字符串形式的！！！
